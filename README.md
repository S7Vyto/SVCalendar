# Installation
1. In terminal go to your project's folder:

   ```js
    cd /User/Developer/YourProjectName 
   ```
  
2. If you haven't pod file, run command:

   ```js
    pod init
   ```
   
3. Open your pod file and write: 

   ```js
    pod 'SVCalendar', '~> 1.0.7'
   ```
   
4. Save file and run command:
   
   ```js
    pod install
   ```

5. That's it!
